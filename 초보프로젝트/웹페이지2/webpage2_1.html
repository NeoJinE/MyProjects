3. git

1.git config --global user(name&email) => 깃허브 계정값 설정.
2.git config --list => 작성값 확인용도.
3.git init => 디파짓토리 생성코드 => 코딩폴더 가장 상위폴더로 만드는게 좋다
4.git add.=> .git의 모든 파일을 선택하여, 디파짓토리에 저장한다.
git add(폴더 및 파일)=> 선택한 파일 및 폴더만 ``
5.git status => 디파짓토리에 저장된 코드를 확인.
6.git commit -m"first commit" => 깃허브에 커밋하는 과정.
7.git remote add origin(깃허브 링크)=>설정한 깃허브 링크에 커밋한다.
8.git push origin master => 깃허브에 커밋한 코드를 푸쉬(저장)

9.git checkout -b(name)=>새로운 branch 선언 git checkout => 해당 이름의 branch로 전환.
